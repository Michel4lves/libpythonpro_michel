Github Actions funcionando.

Tópicos a serem abordados:
1. Git;
2. Virtualenv;
3. Pip.

Para instalar:

console

    py -3 -m venv .venv
    .venv\Scripts\activate
    .venv\Scripts\python.exe -m pip install --upgrade pip
    pip install -r requirements-dev.txt

Para conferir qualidade de código:

console

    flake8
